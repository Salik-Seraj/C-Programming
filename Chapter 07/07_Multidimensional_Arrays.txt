                  ☣️Multidimensional Arrays☣️

An array can be of 2 dimension / 3 dimension / 4 dimension / n dimensions.

A 2 dimensional array can be defined as :

int arr[3][2] = {1,4}
                {7,9}
                {11,22};

We can access the elements of this array as arr[0][0], [0][1] & so on ....


⟫ 2-D Arrays In Memory

A 2d array like a 1-d array is stored in contiguous memory blocks like this :

arr[0][0] arr[0][1] ...
